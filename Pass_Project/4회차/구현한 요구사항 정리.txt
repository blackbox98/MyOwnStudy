1. 메인 화면
 1.1. 시,구,동을 선택하여 동별 아파트 실거래 정보를 검색할 수 있습니다.
 1.2. 아파트를 선택하면 아파트별 실거래 정보를 보여줍니다.
 1.3. 로그인 전 : 로그인, 회원가입 메뉴 출력, 로그인 후 : 로그아웃, 마이페이지 메뉴를 출력합니다.

2. 로그인 화면
 2.1. 아이디와 비밀번호를 입력하는 form과 로그인, 비밀번호 찾기, 회원가입 버튼을 출력합니다.
 2.2. 아이디 저장 기능을 구현하여 편의성을 높였습니다.
 2.3. 비밀번호 찾기 기능은 단순히 아이디와 이메일을 입력받아 비밀번호 찾기를 클릭하면
      로그인 페이지로 이동하며 입력 form 아래에 메시지로 출력해주는 기능입니다.
      등록된 계정인 경우 비밀번호를 출력하고, 등록되지 않은 계정일 경우 등록되지 않았다는 메시지를 각각 출력합니다.
 2.4. 회원가입 버튼 클릭 시 회원가입 form으로 이동합니다.

3. 회원가입
 3.1. 메인 화면과 로그인 페이지에서 회원가입 메뉴를 사용할 수 있습니다.
 3.2. 클릭 시 회원가입 form으로 페이지를 이동하고, 아이디, 비밀번호, 이메일, 이름, 나이를 각각 입력하여 회원가입을 합니다.

4. 마이페이지
 4.1. 회원정보 관리 페이지인 마이페이지에 들어가면 전체 회원 조회, 회원정보 조회, 회원정보 수정, 회원정보 삭제 기능을 사용할 수 있습니다.
 4.2. 전체 회원 조회는 현재 등록된 모든 계정의 정보를 보여주며, 해당 페이지 내에서 회원정보의 수정/삭제는 불가합니다.
 4.3. 회원정보 조회 : 현재 로그인한 계정의 정보를 출력합니다. 해당 페이지에서 회원정보의 수정/삭제는 일어나지 않기 때문에
      모든 정보에 대해 readonly 옵션을 적용하였습니다.
 4.4. 회원정보 수정 : 현재 로그인한 계정의 정보를 출력하며, 해당 페이지에서는 회원정보의 수정 작업이 일어나기 때문에 PK인 아이디를 제외한
      모든 정보의 수정이 일어날 수 있으므로, 아이디에만 readonly 옵션을 적용하였습니다.
 4.5. 회원정보 삭제 : 현재 로그인한 계정의 정보를 출력하고, 수정 작업은 이루어지지 않기 때문에 모든 정보에 대해 readonly 옵션을 적용하였고,
      회원 탈퇴 버튼을 클릭하게 되면 실수를 방지하기 위해 window.confirm() 메서드를 활용하여 한번 더 체크하는 기능을 추가하였습니다.
